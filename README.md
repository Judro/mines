## Installation
install the deb file: [mines_0.3.0_amd64.deb](https://github.com/Judro/mines/releases/download/v0.3.0/mines_0.3.0_amd64.deb)

## Controls
- Movement: `hjkl or ←↓↑→`
- Place flag: `f`
- Open tile: `s`

## Build from source
1. install dependencies
2. `make`
3. `make install`

### Dependencies
- libncurses-devel
